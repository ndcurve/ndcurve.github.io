Chapter 4 - Arrays

Big Variables

Imagine a set of bytes starting from 100 and ending at 140. 
Imagine it as a big variable.
It is a variable starting from 100 and ending at 140.
This variable contains 40 bytes.
A regular variable (number) contains 4 bytes.
This variable can hold 10 variables (40/4 = 10).
Name this big variable as "A".

                    RAM POSITION

                    100             100          104            108           112           .....  136  
                    Variable A = Variable 1, Variable 2, Variable 3, Variable 4, .....  Variable 10.


Variable A Position = 100
Variable 1 Position = 100 + (0x4) = 100
Variable 2 Position = 100 + (1x4) = 104
Variable 3 Position = 100 + (2x4) = 108
...
Variable 10 Position = 100 + (9 * 4) = 136


Javascript Notation

Create Array Variable
let A
A = []

Create Array Variable with Values
let A
A = [5, 8, 5, 2, ...., 16]

Read Array Variable
let b
b = A[0]
b = A[1]
b = A[2]
b = A[3]
b = A[9]

let k = 0
b = A[k]

Write Array Variable
A[0] = 50
A[9] = 800

let k = 0
A[k] = 25



Add new Value to end of Array
A.push(50)

Remove last Value
A.pop()

Array variables count
let c
c = A.length
